A complete reference for agentic AI commands and concepts — from practical Claude Code slash commands to universal patterns that apply across any AI agent system.

- **Part I: Claude Code — Practical Commands**
  Session management, context tools, model/mode control, configuration, extensions, keyboard shortcuts, CLI flags, and bundled skills.

- **Part II: Universal AI Agent Command Glossary**
  The conceptual vocabulary behind agentic AI systems — delegation, tool use, memory, planning, guardrails, and more. Framework-agnostic patterns you'll encounter across any agent platform.
